# 135 Film Splitter

> 🤖 **A Vibe Coding Experiment**
> 
> 本项目是一次**纯粹的 Vibe Coding 尝试**。全流程代码、UI 架构、性能调优以及各种复杂的边缘情况处理，均由用户通过提供直觉反馈和截图，全程使用 AI 编写和迭代完成。

**135 Film Splitter** 是一款专为摄影师与胶片扫描爱好者打造的跨平台桌面软件。它的核心目标只有一个：**将专业扫描仪生成的包含了多张照片的巨型高位深胶片底片扫描件，又快又好地自动切分为单张照片**。

---

## 🌟 支持功能

- **🚀 自动智能切分**
  - **支持多条带混扫**：不论您是一次扫描了一条胶片，还是并排扫描了两条甚至多条胶片，它都能自动识别出每一条胶片的位置。
  - **支持特殊画幅**：内置 135 胶片的物理规则约束，不仅能完美识别标准的 **全格 (Full-frame)** 照片，也能精准捕捉间距更紧密的 **半格 (Half-frame)** 照片。

- **🎛️ 自由的交互微调**
  - **极速丝滑的 UI**：无论是上 GB 还是 16-bit / 32-bit 的 TIFF 巨幅底片，由于采用了代理图架构，您在界面上的拖拽和缩放操作都不会产生任何卡顿。
  - **手工微调**：自动识别后，生成的绿色裁切框支持**鼠标拖拽移动**，您也可以将鼠标悬停在四条边缘和四个角落进行**精确的大小缩放微调**。

- **💎 纯粹的无损画质导出**
  - **绝不偏色，拒绝压缩**：程序在裁切导出时，会直接从高位深原图内存中截取，并且**完全不使用任何会损失画质的压缩手段**。
  - **原图元数据继承**：导出的每一张单帧切片，都会完美继承原图的 `DPI 分辨率` 与最重要的 `ICC 色彩配置文件`。确保导入 Lightroom 或 Photoshop 后立刻呈现原本准确的色彩。

## 📖 使用指南

1. **Load TIFF (加载底片)**
   - 点击左侧控制面板的 `Load TIFF`，选择您的原图扫描件。程序会自动在后台解析并生成代理预览图。
2. **Auto Detect Frames (自动识别)**
   - 加载完成后，点击 `Auto Detect Frames`，程序将瞬间为您找出所有并排的胶片以及它们内部的每一张单帧照片，并在右侧画布生成绿色裁切框。
3. **手动微调 (可选)**
   - 如果遇到极个别识别偏差（如过暗或过曝的区域），您可以直接在右侧画布中：
     - **移动**：拖动画框中心区域。
     - **缩放**：拖动画框的四条边或四个角落。
4. **Export Slices (无损导出)**
   - 调整完毕后，点击 `Export Slices` 并选择一个用于存放单张照片的文件夹。
   - 等待几秒钟，导出完成后会有弹窗提示。此时您可以去目标文件夹查看原汁原味的单帧 TIFF 图片了。
